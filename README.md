CopyFlow — Master Your Text Queue

Turn messy text into clean, paste-ready items in seconds. Automate repetitive typing and streamline your workflow.

Overview

CopyFlow is a productivity tool that takes any messy list of text, cleans it, and converts it into a queue of items ready to paste one by one with F9. Perfect for:

Excel or Google Sheets data entry

Online forms

Repetitive typing tasks

Organizing large text lists

CopyFlow makes repetitive tasks faster, easier, and less error-prone.

Features

Automatic List Cleaning & Splitting
Handles commas, semicolons, pipes, tabs, bullets, newlines, and special characters.

Three Parsing Modes

Lenient: Minimal splitting; best for long notes.

Normal: Splits common separators; ideal for typical lists.

Strict: Forces splits at everything—perfect for serial numbers or tight datasets.

Queue Management

Add, manage, and track items in a live queue.

Undo the last operation to recover accidental changes.

Hotkey Automation

Press F9 to paste the next item automatically.

Optional Auto-Tab mode for Excel or form navigation.

Batch Run

Automatically paste all items with configurable delay.

Panic button stops the batch at any time.

Search & Filter

Quickly find specific items in a large queue.

Persistent Storage

Auto-saves queue to copyflow_master_data.json so you can resume where you left off.

Export Queue

Export items as JSON for reuse or sharing.


Usage

Paste or type your messy list into the input box.

Select a parsing mode (Lenient / Normal / Strict).

Click Process & Add to add items to the queue.

Use F9 to paste the next item anywhere.

Optional: enable Auto-Tab for Excel or form navigation.

Track progress using the progress bar and queue view.

Keyboard Shortcuts
Shortcut	Action
F9	Paste next item in queue
Ctrl+Enter	Process & Add input text
Ctrl+Z	Undo last operation
Known Limitations

F9 automation requires the target window to be focused.

Strict mode may split spaces inside multi-word items.

Currently no tray mode or floating preview HUD (planned for future updates).
